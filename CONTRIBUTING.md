# Contributing

このテンプレートを使ってみて、改善のアイデアがあればぜひ共有してください。

## 歓迎するコントリビュート

- **部署の追加提案** — 「税理士枠」「司法書士枠」「リフォーム職人枠」など、不動産大家業に役立つ役割
- **サンプルデータの充実** — 戸建て・区分・テラスハウスなど、アパート以外の物件サンプル
- **キャラ名の提案** — 「ハンター」「番頭」より直感的な名前案
- **判断軸の改善** — 各部署の `CLAUDE.md` の「判断軸」セクションへの追記
- **誤字・表現の修正** — 細かい改善も大歓迎

## 進め方

### Issue で相談する
- バグ報告・機能要望は [Issues](../../issues) から
- テンプレートに従ってください（`.github/ISSUE_TEMPLATE/`）

### Pull Request を送る
1. このリポジトリをフォーク
2. 機能ブランチを作成（`feat/add-tax-advisor` のような名前で）
3. 変更をコミット
4. PR を作成

## 改変時のお願い

- **個人情報を含めない** — サンプルデータは「実在しそうな架空のもの」に徹してください
- **特定コミュニティ・特定講師色を含めない** — このリポジトリは汎用テンプレートです
- **ファイル名は kebab-case** — `sample-apartment.md` のような形式で
- **Markdown は末尾改行つき** — markdownlint passable に
